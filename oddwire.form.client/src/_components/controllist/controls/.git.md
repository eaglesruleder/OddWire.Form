# Brief - Form Controls

**Scope:** Subsystem  
**Domain:** oddform  
**Status:** Active

---

## What It Is

A control library for the dynamic form renderer. Leaf controls render Bootstrap-backed fields through a shared props contract and emit `onChange(value, key, subkey?)`; layout controls render nested `ControlList`s over a shared or row-scoped `InstanceEntity`; `ControlBase` centralises row layout, hidden gating, and readonly display for leaves that use it.

Controls are mostly stateless. Values live in `InstanceEntity` / parent state; layout controls only keep local UI state such as expanded/open/active tab. A looper is the exception-shaped layout control: its parent value is a row-instance array, and each row renders the same template controls against its own child instance.

---

## Core Loop

```text
Leaf:
receive resolved props
-> ControlBase gates hidden and lays out label + field
-> user edits
-> leaf emits onChange(value, props.param)
-> parent patches InstanceEntity

Layout:
receive controls + instance + onChange
-> manage local UI state
-> render nested ControlList over same instance
-> child edits call same root onChange

Looper:
receive controls + value[]
-> render each row as child InstanceEntity(row)
-> child edits replace that row in value[]
-> optional Add Row appends an empty sparse row
```

Inputs: `CoreControlProps` (`param`, `label`, `value`, `hidden`, `stacked`, `disabled`, `placeholder`, `readonly`, `onChange`, grid classes) plus per-control extras.  
Output: renderable control UI and, for interactive leaves, `onChange(value, param, subkey?)`.

---

## Driving Values

| Value | What drives it | What it does |
|---|---|---|
| `param` | Definition | DOM id/name, update key, and instance identity |
| `label` | Definition, after `resolveLabel` upstream | Text shown beside/above the field |
| `hidden` | Props | Suppresses rendering |
| `stacked` | Props | Switches label-left row to label-above column |
| `disabled` | Props / lookup fallback | Disables the field or option interaction |
| `placeholder` | Props / lookup fallback | Empty field hint or missing-option message |
| `readonly` | Props | `true` renders static text; function renders custom replacement node |
| `rows` | Textarea prop or import overlay | Sets textarea height, defaulting to 3 |
| `valueType` / `keyboardType` | Text field config | Chooses `inputMode` |
| `controls` | Radio/dropdown or layout definition | Options for leaves; child controls for layout |
| looper `value` | Instance overlay | Row-instance array rendered against the looper template |
| `addRows` | Looper definition | Shows the Add Row button |
| `dbOptions` | Resolved upstream by `ControlItem` | Supplies radio/dropdown options and optional fill behaviour |
| `cellClassName` | Definition | Grid span class applied to the `ControlBase` row |
| `className` | Props | Runtime style/class hook |
| `pageLayout` | Root `ControlTab` | Moves tab bar to fixed footer-style root navigation |

---

## Class Responsibility

| Class | Owns |
|---|---|
| `ControlBase` | Label+field row, hidden gate, readonly field replacement, `cellClassName` |
| `ControlText` | Static separator/value block with its own hidden gate |
| `ControlTextField` | Single-line string input and keyboard hint |
| `ControlTextArea` | Multi-line string input, forced stacked layout |
| `ControlCheckbox` | Boolean checkbox row with its own hidden gate |
| `ControlImage` | Static image display from URL/URN value |
| `ControlRadio` | Inline single-select radio group or missing-option text |
| `ControlDropdown` | Searchable single-select combobox with filtered option menu |
| `ControlButton` | Bootstrap action button used by popups and settings editors |
| `ControlCollapsible` | Local expanded state, sticky header, optional right subtitle, nested `ControlList` |
| `ControlTab` | Section tabs, active section state, inline sticky or root fixed tab bar |
| `ControlPopup` | Button opener, Bootstrap modal, `StripLayout` modal header, nested content or `ControlList` |
| `ControlLooper` | Repeated row renderer; stores row instances in its own value and appends sparse rows |
